Function that lets the light perform one breath cycle in the specified color.

It uses this_thread::sleep_for to accomodate for the time an HueLight::alert() needs The hue ranges from 0 to 65535, whereas 65535 and 0 are red, 25500 is green and 46920 is blue. The saturation ranges from 0 to 254, whereas 0 is least saturated (white) and 254 is most saturated (vibrant).

hueThe hue of the color
satThe saturation of the color
